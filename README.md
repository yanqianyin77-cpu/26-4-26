Kotoba Journal
一个基于 Tkinter + SQLite 的日语单词学习桌面应用。

它把 课文分词、词汇收录、复习、随机自测、错题回看 和 学习记录 放在一个本地项目里，适合做个人日语词汇手帐。

主要功能
课文手札：粘贴日语课文，自动分词、提取重点词
词汇本：手动新增、快速收录、编辑词条、筛选和搜索
复习计划：根据学习状态生成待复习词
随机自测：随机抽题查漏补缺
错题本：自动收集答错词，支持专项练习
学习报告：查看掌握情况、错题数量、学习趋势
设置中心：主题、字体、复习阶段、备份恢复
技术栈
Python 3
Tkinter
SQLite3
Janome
pandas（可选，用于 Excel 导入）
运行环境
建议环境：

Windows 10 / 11
Python 3.11 或 3.12
安装依赖
先安装基础依赖：

pip install janome
如果你需要 Excel 导入功能，再安装：

pip install pandas openpyxl
运行项目
在项目目录执行：

python kotoba_journal.py
项目文件说明
kotoba_journal.py
主程序入口，包含界面、数据库操作、复习逻辑和文本分析逻辑

kotoba_journal.db
本地 SQLite 数据库

logs/
运行日志目录

backups/
数据库备份目录

使用流程建议
第一次使用时可以按这个顺序：

打开 课文手札
粘贴一段日语课文
分析课文中的词
把需要的词加入 词汇本
去 复习计划 开始当天复习
用 随机自测 和 错题本 查漏补缺
注意事项
没有安装 Janome 时，分词能力会明显下降
Excel 导入依赖 pandas
数据默认保存在本地，不依赖服务器
建议定期在 设置中心 做数据库备份
当前状态
这是一个持续迭代中的个人学习工具项目，界面、交互和复习逻辑还在不断优化中。
